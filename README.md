## Chat attachments (R2)

Chat file uploads land in a Cloudflare R2 bucket. To run locally:

1. Create an R2 bucket and an API token (Read & Write, scoped to that bucket).
2. Apply the CORS policy from `docs/superpowers/specs/2026-04-13-r2-attachments-design.md` (CORS section).
3. Fill `R2_*` in `.env.local`.

The `attachments` table tracks each upload; `anthropic_file_id` caches the
Anthropic Files API id so we never re-upload bytes after the first chat turn.
